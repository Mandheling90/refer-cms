========================================
  refer-cms 운영 환경 실행 가이드
========================================

[포트] 3007

----------------------------------------
환경 변수
----------------------------------------

- 개발: .env.local
- 운영: .env.production (빌드 시 자동 적용)

운영 API URL 확정 시 .env.production 수정 필요

----------------------------------------
실행 순서
----------------------------------------

# 1. 의존성 설치
npm install

# 2. 프로덕션 빌드 (.env.production 자동 적용)
npm run build

# 3. 프로덕션 서버 실행
npm run start
